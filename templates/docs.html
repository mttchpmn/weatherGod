WeatherGod API

A Python based API designed to aggregate and propagate the individual sources of weather information for Queenstown, New Zealand. Designed for operational usage in the adventure tourism industry. All information is presented courtesy of its respective source, and retains its original copyright.

Usage of this API is only with express and explicit permission.

Email matt@peakpixel.nz for data enquiries.

github.com/mttchpmn

Rain Radar

River Flow

weathergod.herokuapp.com/api/v1.0/riverflow

Returns a JSON object containing URLs for river flow gifs. Sourced from the Otago Regional Council Website.

Example Response

}
